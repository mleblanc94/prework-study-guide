CSS

  • Enter your CSS notes here

  • A padding indicates how much space we want around the content inside an element.

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add . - adds all files to be committed that have been changed
  • git commit -m "" commits the changes to git as well as adds a message for the pull request
  • git push origin (your branch) - creates the pull request that has to be accepted on GitHub
  • Create files using git touch (name of file)

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is used to repeat a process over and over again
  • A loop is useful for sorting through items in an array
